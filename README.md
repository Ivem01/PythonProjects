# PythonProjects

# 1. Spotify Music Data Analysis

Python data analysis project using a Spotify data set of 114,000 songs. Analyzes features like tempo, popularity, danceability and genre and visualizes the outputs with the graphs.

Features:
- Statistical Analysis: Calculates the averages of tempo, popularity, duration, energy and loudness
- Data Manipulation: Uses Pandas library to process and clean the dataset
- Data Visualization: Uses Matplotlib library to generate histograms, bar charts and pie charts to reveal the different distributions in the data set

# 2. CLI App For To-Do List
   
A simple command-line python app designed to manage tasks using a JSON file. Users can add, remove, show, search, mark as done and filter by status with persistent data storage.

Features:
- Persistent Storage: Uses JSON serialiation to save tasks locally, so data is never lost and remains even after closing the program
- User-Friendly Interface: Simple and easy to use command-line interface
- Data searching: Implemented a built-in keyword search to find specific tasks in the list.
- Status Filtering: Ability to filter tasks based on the statuses: Done and Pending, for better organization
- Error Handling: Basic error handling to ensure correct user input

# 3. Simple Weather App

A simple interactive web app built with Streamlit that shows the current weather of any city that the user input using the OpenWeatherMap API.

Features:
- Live Data: Uses the OpenWeatherMap API to fetch information about the current weather conditions
- Geographic Location: Uses Pandas library to process the coordinates and the streamlit function maps to locate the exact location of the city
- Data conversion: Processes the raw JSON data into Metric units like Celsius
- Error Handling: Basic error handing to tell the users if the city is not found

# 4. Food Prices in Myanmar Dashboard

An interactive and simple dashboard about food prices in Myanmar based on the dataset by World Food Programme and built with Streamlit, uses Matplotlib and Pandas to visualizes and handle the data.

Features:
- Financial Analysis: Calculates the price of the prouducts over the years, compares with other regions and calculates the percentage change between the latest price and previous price
- Filtering Options: Ability to filter commodity, region and choose the currency (MMK or USD) to view the specific results
- Geographic Location: Visualizes the market locations in the filtered regions using the coordinates
- Data visualization: Uses Matplotlib and Pandas libraries to process the data and to plot the average prices over the years to show the price trend
